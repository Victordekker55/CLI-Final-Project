# CLI-Final-Project
My Final Project in CLI. Making a CLI Menu interface

In my program python is the used language with the downloadable libary curses. 
The purpose is a neat menu with a login system.
To install type in cmd promt or powershell "pip install windows-curses"

1. Clone git repo
```cmd

```

