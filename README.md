# CLI-Final-Project
My Final Project in CLI. Making a CLI Menu interface
## Description
In my program python is the used language with the downloadable libary curses. 
The purpose is a neat menu with a login system.
To install type in cmd promt or powershell "pip install windows-curses"



## Technology/Languages/Built with

I have used python for all the parts in the code. Frontend aswell as backend was built with the help of the libary windows curses. Windows cures is a module used for windows terminal to show text colour etc. 

- Python
- Windows curses

## Requirements


- Python 3.7+

- Time module
- Curses module
- Random module

## Installation



1. Clone git repo
```cmd
    git clone https://github.com/Victordekker55/CLI-Final-Project
```
2. Installera Curses
```cmd
pip install windows.curses
```

## Code conventions

The code will not run correctly if resource.txt is not in the same file destination as main.py

## Usage
You can use this code for a template for a more advanced login system aswell as a template for CLI gui. Also could be used for fun with new users experimenting with curses.
